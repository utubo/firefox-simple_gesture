Experimental - Simple Gesture
SimpleGesture.doCommand()

You can use "SimpleGesture.doCommand()" to execute Simple-Gesture's command on custome-gestures.
e.g.
SimpleGesture.doCommand('reload');

Note1:
TAKE CARE NOT TO MAKE AN INFINITE LOOP.

Note2:
This API can't change a target of some commands.
e.g.
// when
// TAB-A = the current tab
// TAB-B = the next tab

// show TAB-B
SimpleGesture.doCommand('nextTab');

// reload TAB-A (NOT TAB-B)
SimpleGesture.doCommand('reload');

// Simple-Gesture can't realod TAB-B on this version, sorry.

Note3:
This API is executed asynchronously.

Note4:
JavaScript will be suspended, when page is closed or reloaded.
e.g.
You can't make a gesture to "Reload and scroll to bottom".

If you want to do something that is not simple, then execute "browser.runtime.sendMessage(...)" or create a custom event with custom-gesutre, and catch it with greasemonkey or others.
